== Installation ==
<p>
В административной панели:<br/>
<ol>
<li>
Плагины->Добавить новый->В поле ввода "Поиск плагинов"-> Ввести "VDZ Yandex Share".
</li>
<li>
Установить найденный плагин.
</li>
<li>
Активировать плагин VDZ Yandex Share
</li>
<ol>
</p>
<p>
FTP:<br/>
<ol>
<li>
Распаковать архив с плагином
</li>
<li>
Загрузить папку с плагином в директорию /wp-content/plugins/ на вашем сервере
</li>
<li>
Активировать плагин VDZ Yandex Share в разделе "Плагины" административной панели WordPress
</li>
<ol>
</p>
